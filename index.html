Sobre Mim

Me conheça melhor.

Desenvolvedor Full-Stack com experiência no desenvolvimento de aplicações web modernas, atuando tanto no front-end quanto no back-end. Trabalho com foco em performance e boas práticas de desenvolvimento, entregando soluções eficientes.

Atualmente, utilizo principalmente PHP com o framework Laravel para o desenvolvimento de APIs e sistemas no lado do servidor. No front-end, construo interfaces modernas e responsivas com HTML, CSS, JavaScript e TailwindCSS, garantindo uma ótima experiência para o usuário final.

Tenho experiência com bancos de dados relacionais, especialmente MySQL, realizando desde modelagem até otimizações de queries para melhorar o desempenho das aplicações.

Além disso, possuo conhecimentos em outras linguagens como Java e Python, e também em Docker para gerenciar aplicações, o que me permite atuar com mais flexibilidade em diferentes tipos de projetos e tecnologias.
